# Páginas públicas do Study Hub

Esta pasta está pronta para GitHub Pages.

## Publicação recomendada

1. Suba o repositório para GitHub sem arquivos secretos.
2. No GitHub, abra **Settings > Pages**.
3. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: `main`
   - Folder: `/docs`
4. Salve e aguarde a publicação.

## URLs esperados

- Home: `https://vilitor.github.io/study-hub/`
- Política de Privacidade: `https://vilitor.github.io/study-hub/privacy.html`
- Termos de Uso: `https://vilitor.github.io/study-hub/terms.html`

Use esses URLs na tela de consentimento OAuth do Google depois que o GitHub Pages estiver ativo.
